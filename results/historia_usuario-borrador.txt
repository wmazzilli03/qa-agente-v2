Feature: Inicio de sesión de usuario

  @login_exitoso
  Scenario: Login exitoso con credenciales válidas
    Given El usuario se encuentra en la página de login
    When El usuario ingresa usuario "standard_user"
    And El usuario ingresa contraseña "secret_sauce"
    And El usuario hace clic en "Login"
    Then El usuario debe ser redirigido al inventario de productos
    And El inventario debe mostrar los productos disponibles

  @login_contraseña_incorrecta
  Scenario: Login fallido con contraseña incorrecta
    Given El usuario se encuentra en la página de login
    When El usuario ingresa usuario "standard_user"
    And El usuario ingresa contraseña "password_incorrecta"
    And El usuario hace clic en "Login"
    Then Se debe mostrar un mensaje de error
